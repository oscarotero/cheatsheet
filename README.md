# Cheatsheet

## Characters

Description | Char
------------|-----
Em dash | `—`
En dash | `–`
Ellipsis | `…`
Arrows | `←` `↑` `→` `↓`
Multiply | `×`

## SVG

Embed in CSS:

```css
url('data:image/svg+xml;utf8,<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 1080 1080" fill="%23000">...</svg>')
```
